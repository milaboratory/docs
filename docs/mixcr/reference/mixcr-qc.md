# `mixcr qc`

Perform quality control checks on resulting `.vdjca` or `.clns` files.

## Command line options 

```
mixcr qc
    [--json]
    [--print-to-stdout]
    [--print-warn]
    [--force-overwrite]
    [--no-warnings]
    [--verbose]
    [--help]
    input.(vdjca|clns|clna)
    [output.(txt|json)]
```

`--json`
: Output result in json format

`-f`, `--force-overwrite`
: Force overwrite of output file(s).
